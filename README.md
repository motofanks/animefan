# FANKS - ガールズバンドクライ ファンサイト

このプロジェクトは、アニメ「ガールズバンドクライ」のファンサイトのデモンストレーションです。
Cloudflare Pages を使用してホストされています。

## 機能

- ホームページ：作品情報の表示
- プロフィールページ：ユーザーの投稿機能
- 投稿の表示：最新の投稿を表示

## 技術スタック

- HTML
- CSS
- JavaScript（バニラ）

## デプロイ

このプロジェクトは Cloudflare Pages を使用してデプロイされています。
